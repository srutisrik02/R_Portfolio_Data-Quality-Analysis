# Portfolio project of Data Quality Analysis of Stimulated Field Survey Data

## Overview

This project demonstrates analysis and visualisation of stimulated field survey data using R.

## Objectives

- Conduct common data quality checks and identify quality flags
- Calculate summary statistics
- Create visualisations

## Libraries used in R
- dplyr
- tidyverse
- ggplot2
- lubridate
- forcats
